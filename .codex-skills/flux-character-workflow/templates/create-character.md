# Create Character Prompt Template

Use for first-pass character exploration. Keep all variants tied to one stable visual specification.

```text
[Character name], [character type], [precise face or head shape], [skin/fur/surface tone with HEX colors], [eye shape and color], [hair/fur/head silhouette], [body proportions], wearing [base outfit], with [accessories and distinctive marks].

Create a clean canonical character design for future reference images.

Keep consistent across all variants:
- facial structure
- skin/fur/surface tone and HEX palette
- eye shape and color
- hair/fur/head silhouette
- body proportions
- base outfit design
- accessories
- illustration style

Change only:
- small expression differences
- minor pose differences
- small silhouette exploration within the same design

Composition:
- full body front view
- neutral standing pose
- centered subject
- plain light background
- no extra objects

Lighting and style:
- even studio lighting
- [illustration style]
- clean readable silhouette
- palette: [HEX colors]
```

After generation, do not mark a result as canonical until the user explicitly approves it.
