# Create Character Sheet Prompt Template

Use after a canonical design has been approved.

```text
[Character name and precise visual description from character.md].

Preserve the exact character identity, facial structure, skin or fur tone,
eye shape and color, hairstyle or fur silhouette, body proportions,
outfit design, accessories, and illustration style from the reference images.

Change only:
- create a neutral technical character sheet
- show the requested views: [front / three-quarter / side / back / expressions]
- keep all views visually consistent

Keep unchanged:
- all HEX colors from the Character Pack
- facial structure and head silhouette
- outfit cut, pattern, and accessories
- body proportions
- line style and rendering style

Composition:
- orthographic or low-perspective technical presentation
- plain solid background
- evenly spaced views
- no props unless listed as canonical accessories

Lighting and style:
- even studio lighting
- no atmospheric effects
- clean lines
- clear detail visibility
```

If the sheet introduces identity drift between views, create separate angle references by sequential editing from the canonical image.
