# Experiment Log Entry Template

Use for controlled FLUX character experiments.

```markdown
## Experiment: [short title]

Date:
Goal:
Character version:
Model:
Reference images:
Prompt:
Seed:
Result:
Consistency issues:
What worked:
What failed:
Decision:
```

Change only one primary variable per experiment:

- reference set
- preservation block wording
- model
- seed
- edit order
- number of changes in one step
